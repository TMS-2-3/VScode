============================================================
1. ゲーム概要
============================================================

仮タイトル: 優秀なヒーラー(仮)
ジャンル: 2D見下ろし型の味方管理・支援バトルRPG
操作キャラ: 白魔法士アルジュナ・フィナルド(名前・姓・性別・一人称は開始時に設定可能)
現在の実装形式: HTML + CSS + JavaScript + Canvas

基本コンセプト:
主人公は前線で直接戦うのではなく、後方味方を支援する白魔法士。
勇者、モンク、黒魔法士の3人は自動で戦うが、調子や攻防方針によって行動が荒くなったり弱気になったりする。
プレイヤーは回復、シェルト、ショック、指示、アイテム、ターゲット指定を使い、癖の強い味方を死なせずに勝たせる。

面白さの中核:
- ただHPを回復するだけではなく、味方の調子、攻防メーター、位置取り、味方の範囲攻撃の誤爆まで管理する。
- 調子が低いと弱くなり、高いと強くなるが、上がりすぎると慢心して被ダメージが増えたり、勝手に必殺技を撃ったりする。
- プレイヤーの指示が必ず通るわけではなく、調子が極端な味方は指示を無視することがある。
- 味方は自動行動なので、プレイヤーは「戦う人」ではなく「危なっかしい味方を勝たせる人」になる。

現段階のプレイフロー:
1. プロフィール設定
2. 町の導入会話
3. 依頼所前で3人と合流
4. 依頼所の近くでEキーを押す
5. ストーリー依頼 / フリー依頼を選択
6. 依頼を選択
7. 出発前確認
8. 戦闘
9. 敵全滅でクリア

6. 町・施設・依頼
============================================================

町の形式:
- 現在はWASDでフィナルドを操作して町を歩く方式
- 仲間合流後は、味方3人がフィナルドの移動軌跡を順番に追いかけて付いてくる
- 施設は近くまで移動してEキーで利用する。町マップ上の施設を左クリックで選択する操作は使わない

施設:
- 依頼所: 依頼選択、戦闘開始の入口
- 鍛冶屋: 武器・防具の作成/強化予定
- アクセサリ屋: アクセサリ関連予定
- 宿屋: 回復、失敗時の強制宿泊予定
- 研究所: スキル解放予定

依頼所の現行フロー:
1. 依頼所の近くでEキー
2. ストーリー依頼 / フリー依頼を選択
3. 依頼を選択
4. 出発前の確認
5. 戦闘開始

現在ある依頼:
- ID: story_tutorial_001
- 種類: ストーリー依頼
- ランク: D
- 名前: 仮依頼: 町外れの魔物討伐
- 目的: 敵を全滅させる
- 敵情報: 魔物、小術師、大魔物
- 戦闘ID: tutorial

ランク進行方針:
- ランクは D / C / B / A / S
- そのランクのストーリー依頼をすべてクリアすると次ランクへ上がる予定
- フリー依頼は稼ぎ・素材・ビルド調整用になる想定

失敗時方針:
- 報酬なしで町に戻る
- 宿屋に一晩強制的に泊まる

7. 戦闘の基本仕様
============================================================

戦闘形式:
- 2D見下ろし型
- Canvas描画
- 味方CPは自動行動
- 主人公はWASDで戦場を移動しながら支援。詠唱中は移動入力不可
- 敵全滅で勝利
- 初期敵を全滅させると左側から小さい前衛敵が3体増援として出現する
- 増援も倒すとクリア

フィールド:
- 上下左右に木の帯があり、戦闘フィールドを狭めている
- 左右の非移動領域にも木が描かれる
- 中央は草地
- 戦闘座標は battleSpatialScale = 0.9 で調整されている

行動間隔:
- 何かの行動後、最低0.2秒の隙がある
- 詠唱中は移動入力不可
- 詠唱中に通常ダメージを受けても詠唱は続行する
- ノックバック、引き寄せ、強制TPなど位置を動かす効果を受けた場合は詠唱中断
- プレイヤー操作による詠唱キャンセルはできない
- 中断された場合でもCD短縮はなく、詠唱中断時から満額CDに入る
- 詠唱が完了した場合は、詠唱完了時から満額CDに入る
- アルジュナの詠唱型必殺技は、詠唱完了または中断時に必殺ゲージを消費する

行動CD:
- 基礎行動CDは6秒
- 味方CPは攻防メーターの activeCommandBias によって行動CDが変わる
- 敵も基礎行動CD6秒を使う
- CPの通常攻撃CDではなく、共通の行動CDが行動テンポとして扱われる

スキル選択AI:
- 行動CDが終わった時、使用可能なスキル候補からランダムに選ぶ
- 射程外のスキルも候補に入る
- 選ばれたスキルが射程外なら、射程内まで移動してから発動する
- 発動後は行動CDを待ち、待機中は適正距離へ戻る
- 候補がない時は0.2秒後に再判定する
- この0.2秒再判定では、調子による攻防メーター自動変更は起きない

ターゲット指定:
- 何も構えていない時、敵を左クリックすると優先ターゲット指定
- 同じ敵をもう一度左クリックすると解除
- 指定された敵が倒れると自動解除
- 味方CPは指定ターゲットを最優先で狙う

予兆線・予兆範囲:
- 敵攻撃には赤系の予兆が出る
- 敵の移動に合わせる必要がある予兆は追従する
- 味方側や支援側の範囲は別色で表示する
- 味方CPは行動/詠唱/硬直中でない場合、敵の予兆範囲から回避する
- 回避中でも、射程内にある近づかないスキルは使用できる
- 回避中は接近スキルや自動必殺技は使わない

味方への誤爆:
- 敵を対象にするスキルは味方を狙わない
- ただし攻撃者の調子が80%以上なら、範囲攻撃や発射物は範囲内の味方にも敵と同じように命中する
- 調子80%未満: 味方には攻撃もデバフも当たらない
- 調子80～89%: 味方へのダメージ30%、デバフなどの効果時間50%
- 調子90～100%: 味方へのダメージ80%、デバフなどの効果時間100%

シールド:
- シールドはHPバーに水色で重ねて表示
- シールドはHPより先にダメージを受ける
- シールドが重なった場合、耐久値を加算する
- シールド量の上限は最大HPの100%
- シールドは内部的に付与ごとの個別スタックとして管理する。表示は合計値
- 複数シールドの持続時間が違う場合、各シールドはそれぞれの残り時間で個別に消える
- ダメージを受けた時は、残り時間が短いシールドから先に消費する
- シールドが存在している間、調子を毎秒+0.3獲得する

8. 操作方法
============================================================

共通:
- 左クリック: 構え中のスキル/指示/アイテムを確定、またはターゲット指定
- 右クリック: 構えキャンセル
- Esc: 三本線のシステムメニューを開閉。町パネルが開いている場合は町パネルを閉じる
- Space: スキルページ1/2の切り替え
- 1/2/3/4: 必殺技

必殺技キー:
- 1: ウルペス
- 2: リハス
- 3: スシア
- 4: アルジュナ

スキルページ:
- 1ページにつき左5枠 + 右端に固定必殺技枠
- 左5枠のキーは Q / E / R / F / G
- 右端の必殺技は常に4
- Spaceでページ1/ページ2を切り替え
- 通常スキルと指示スキルは共通の10枠に自由に配置できる

アイテム:
- C / V / B でアイテムスロットを構える
- 現在Cにポーションx3、V/Bは空
- アイテム構え中に左クリックでカーソル上の味方へ使用
- 右クリックでキャンセル。Space/別スキル/別アイテムでは構え変更
- ポーションは対象の最大HP25%回復
- 回復量が0の場合は消費しない

10. ステータスと共通計算
============================================================

基礎ステータス:
味方全員の基本値は以下。
- HP: 500
- MP: 300
- 攻撃力: 100
- 魔力: 100
- 防御力: 100
- 魔法防御力: 100

敵:
- 現在、敵の防御/魔防は基本100
- HPや攻撃力は敵種ごとに異なる

MP自然回復:
- デフォルトは最大MPの5%/秒
- 最大MP300なら毎秒15回復
- HP/MP再生率はマイナス値も許容する方針

会心:
- 基礎会心率: 10%
- 基礎会心ダメージ: 50%
- 会心が発動すると、ダメージに 100% + 会心ダメージ を掛ける。基礎値なら150%
- ダメージ系の通常攻撃、スキル、必殺技は原則すべて会心判定を行う。発動しないものは個別に明記する
- 会心率が100%を超えた場合、超過分の50%を会心ダメージに還元する
- 例: 会心率120%なら、超過20%の半分で会心ダメージ+10%、基礎50%なら60%になる
- 会心ダメージの下限は0%

ガード:
- 味方CPの基礎ガード率: 20%
- アルジュナのガード率: 0%
- ガード時の被ダメージ倍率: 0.5倍
- 詳細UIではガード軽減率50%と表示
- ガード率が100%を超えた場合、実判定と表示は100%まで
- 超過分は30%につき10%のガード軽減率に変換
- ガード軽減率の上限は85%

ステータスバフの考え方:
- 基礎ステータス = デフォルト値 + 実数値バフ/装備補正
- 割合バフは基礎ステータスに対して加算
- 最終ステータス = 基礎ステータス + 基礎ステータス * 割合補正合計
- 割合バフ同士は乗算しない

下限:
- HP: 0以上、最大HPは1以上
- MP: 0以上、最大MPは0以上
- 攻撃力: 0以上
- 魔力: 0以上
- 防御力/魔法防御力: 現実装では1以上
- 移動速度: 0以上
- 会心率: 0以上
- 会心ダメージ: 0%以上

詠唱速度:
- 最大100%
- 100%なら詠唱時間は実質0.0001秒
- マイナスの場合は詠唱時間が増える
- 例: -50%なら1.5倍、-200%なら3倍

物理/魔法強化・耐性:
- 内部処理として physicalDamageBoost / magicDamageBoost / physicalDamageResistance / magicDamageResistance がある
- UIでは大きく見せない方針
- 装備や状態効果で使う可能性あり

11. ダメージ計算
============================================================

技側の基礎ダメージ:
基礎ダメージ = 技の基本ダメージ + 参照ステータス * スケール

現在の処理順:
1. 基礎ダメージを作る
2. 会心補正
3. 防御/魔法防御による軽減
4. 与ダメージ率・被ダメージ率・物理/魔法強化/耐性・特殊補正を加算的にまとめる
5. 属性倍率。現在は無属性のみなので実質1倍
6. 味方巻き込み補正
7. ガード判定とガード軽減
8. シールド吸収
9. HPダメージ
10. 遅延ダメージ処理など

補正の考え方:
- 与ダメージ率や被ダメージ率は加算型
- 例: 与ダメ+20%、被ダメ+40%、別補正-10%なら、1 + 0.2 + 0.4 - 0.1 = 1.5倍
- 属性系を再導入する場合は乗算型にする予定

防御/魔防:
- 物理ダメージは防御力を参照
- 魔法ダメージは魔法防御力を参照
- 現在は防御100を基準とした軽減式で動く

12. 調子メーター
============================================================

対象:
- アルジュナ以外の味方CPが持つ
- 0～100%
- 初期値は35%
- 基準は50%

役割:
- HPが高く保たれたり、活躍すると上がる
- HPが削られる、被ダメージが大きいと下がる
- 少し調子に乗っている時が最もパフォーマンスが出やすい
- 上がりすぎると慢心し、被ダメージが増えたり必殺技を勝手に使う
- 下がりすぎると萎縮して火力が落ちる

自然変動:
現在HP割合で毎秒変動する。
- HP85%以上: +1/s
- HP70%以上85%未満: +0.5/s
- HP60%以上70%未満: +0.2/s
- HP50%以上60%未満: -0.3/s
- HP40%以上50%未満: -0.5/s
- HP30%以上40%未満: -0.8/s
- HP20%以上30%未満: -1/s
- HP10%以上20%未満: -1.5/s
- HP10%未満: -3/s

調子上昇:
- 与ダメージによる上昇は、敵の最大HPに対するダメージ割合を元にする
- 与ダメージによる上昇量は現在0.85倍に調整済み
- 敵撃破時、攻撃者に+3
- 命中距離が200px以上で距離ボーナスが入り始め、420px以上で最大1.5倍
- 1つのスキルで敵に2体以上同時命中した場合、3 + 命中数 の固定ボーナス
- 被ダメージしていない時間が3秒を超えると、獲得する調子に倍率がかかり、18秒以上で最大1.3倍
- 調子90%以上では、すべての調子上昇値が30%減少

調子低下:
- 被ダメージによる低下は、自分の最大HPに対する被ダメージ割合を元にする
- 被ダメージによる低下量は1.1倍に調整済み
- 攻防メーターでさらに増減する

回復による調子:
- 回復したHP割合を元に調子が上がる
- 回復による上昇量は以前より8%低下済み
- 被ダメージから3秒以内のヒールは追加+2

与ダメージ率:
- 0～10%: 0.3倍
- 50%: 1倍
- 90%: 1.3倍
- 100%: 1.3倍
- 間は線形補間

被ダメージ率:
- 70%以下: 1倍
- 80%: 1.1倍
- 90%: 1.25倍
- 100%: 1.4倍
- 間は線形補間

必殺技との関係:
- 戦闘中、味方の必殺ゲージは自然増加で毎秒0.6%増える。敵1体にダメージまたはデバフを与えた時、味方1人に回復・シールド・バフを与えた時に、発動者の必殺ゲージが4%増える。必殺技による効果では必殺ゲージは増えない。
- 調子50%以下では味方CPの必殺技を発動できない
- 調子85%以上で、必要必殺ゲージ量に達していれば低確率で自動発動
- 調子95%以上で、必要必殺ゲージ量に達していれば自動発動
- 調子による自動発動時、敵に与えるダメージと効果は60%
- 自動発動時でも、味方に与えるダメージと効果は味方巻き込み補正で別計算

調子による廃止済み補正:
- ガード率補正なし
- スキルCD補正なし
- 詠唱時間補正なし

13. 攻防メーター
============================================================

対象:
- アルジュナ以外の味方CP
- 左が防御、右が攻撃
- 5段階: -2 / -1 / 0 / +1 / +2


スキル待機列:
- 味方CPと敵は、各ユニットごとにスキル待機列を持つ
- 戦闘開始直後や、まだ待機列に使えるスキルがない場合は、従来通り使用可能なスキルからランダムに選ぶ
- 一度使ったスキルは、専用CDがある場合はそのCD終了時、通常攻撃など専用CDがない場合は行動CD終了時に待機列の末尾へ入る
- 2回目以降の行動判断では、待機列の先頭側から見て、現在発動できる最初のスキルを使う
- 待機列の先頭スキルがMP不足、対象なし、射程外移動中などでまだ発動できない場合は、詰まらないように後続の発動可能スキルを見る
- 通常攻撃も待機列に入り、他のスキルと同じく待機列順の候補として扱う
反映タイミング:
- メーターの表示値 commandBias はすぐ変わる
- 実際に効果が出る activeCommandBias は、そのCPが次の行動を開始する時に反映される

効果:
段階 | ガード率 | 行動速度 | 調子上昇 | 調子減少 | 与ダメージ率 | 被ダメージ率
-2 防御最大 | 3.0倍 | 40% | 0.6倍 | 1.5倍 | 0.8倍 | 0.8倍
-1 防御寄り | 1.8倍 | 63% | 0.8倍 | 1.2倍 | 0.9倍 | 0.9倍
0 通常 | 1.0倍 | 100% | 1.0倍 | 1.0倍 | 1.0倍 | 1.0倍
+1 攻撃寄り | 0.6倍 | 125% | 1.0倍 | 1.0倍 | 1.1倍 | 1.15倍
+2 攻撃最大 | 0倍 | 167% | 1.1倍 | 0.9倍 | 1.2倍 | 1.3倍

詳細ステータスの行動速度は、行動速度 = 100% / 行動CD倍率 で表示する。
内部処理では従来通り、基礎行動CDに倍率をかけて実時間を決める。

基礎行動CD6秒での実時間:
- -2: 15秒
- -1: 9.6秒
- 0: 6秒
- +1: 4.8秒
- +2: 3.6秒

適正距離:
前衛/後衛で距離が異なる。
- -2: 前衛160px / 後衛270px
- -1: 前衛100px / 後衛250px
- 0: 前衛65px / 後衛230px
- +1: 前衛50px / 後衛210px
- +2: 前衛40px / 後衛200px

調子による適正距離補正:
- 調子75～100%で距離倍率が1倍から0.6倍へ下がる
- 調子30～0%で距離倍率が1倍から1.4倍へ上がる
- 敵は基本的に0段階の距離を使う

調子による自動変更:
- 各CPは戦闘開始後5回行動するまでは自動変更しない
- 実際に行動を開始するタイミングでのみ判定
- 候補なしの0.2秒再判定では動かない

自動変更確率:
- 調子0～20%: 10%で-2、25%で-1
- 調子21～40%: 15%で-1
- 調子65～80%: 15%で+1
- 調子81～100%: 10%で+2、25%で+1
- 現在は同じ調子帯の判定を1つの乱数で処理する

指示無視:
- 調子30%以下: 攻撃寄りにする指示を30%で無視
- 調子5%以下: 攻撃寄りにする指示を90%で無視
- 調子70%以上: 防御寄りにする指示を30%で無視
- 調子95%以上: 防御寄りにする指示を90%で無視
- 全体指示は味方ごとに個別判定

14. スキルセット・ロードアウト
============================================================

基本:
- パッシブは1つ装備
- パッシブは必ず装備する
- 必殺技も必ず1つ扱う想定
- 通常攻撃は将来的に武器で変わる

アクティブスキル枠:
- 基本キャラ: 5枠
- アルジュナ: 10枠
- アルジュナの10枠には通常スキルと指示スキルを混在できる
- 画面では5枠ずつページ1/2に表示
- 必殺技は右端固定枠で、10枠には含めない

初期ロードアウト:
- アルジュナ: ヒルメント / ショック、ヒール、シェルト、防御指示、攻撃指示、防御陣形、攻撃陣形
- ウルペス: 慣れた剣捌き / ウルペスラッシュ、ウルペススラッシュ
- リハス: 痛みなし / 振り払い、ランウェーブ
- スシア: あったまってきたよ！ / 魔力弾、ファイア（ボンバーは初期から外し、スキルデータとして残す）

スキル入手方針:
- スキルはランク別の結晶からランダム獲得
- 通常攻撃スキルは結晶からは獲得しない予定
- 結晶はそのランクの敵を倒した時に一定率で入手
- 素体を持っていないスキルは選出率を少し上げる案
- 強化が完了したスキルは選出から外す案
- そのランクのスキルをすべて強化済みなら結晶はお金になる予定

15. アルジュナのスキル
============================================================

パッシブ: ヒルメント
- 味方に与えた実回復量の30%分、自身を回復する
- 味方の最大HPを超えて回復されなかった超過分は含まれない
- 自分自身への回復では発動しない
- 常備状態アイコンは「癒」
- アルジュナは現在フィールド上の通常ユニットとして扱う

ショック
- キー: 任意枠(Q/E/R/F/Gのどこか)

- 属性系は現在入れない方針

28. 共同開発時の確認ルール案
============================================================

大きな仕様変更時:
- 仕様メモまたは詳細.txtに反映する
- 実装後、aimemoに「何を変えたか」を短く残す
- 数値だけ変えた場合も balance.js と資料のズレに注意する

データ追加時:
- ゲーム実行用は data/*.js
- 人間確認用は memo.xlsx
- 両方を同期する必要がある場合は、どちらを先に正とするか決める

バグ修正時:
- 発生条件
- 原因
- 直したファイル
- 残る不安
をメモしておくと後で追いやすい

以上。
・戦闘中の詳細ステータスでは、状態アイコンやスキルアイコンにカーソルを合わせると説明ツールチップを表示する。説明文は今後データ側に追加する。
・味方AIの詠唱・予兆付き行動も、行動決定時に攻防メーターを確定し、行動CDと個別スキルCDは詠唱完了または中断時から開始する。通常被ダメでは中断しない。
・味方AIの詠唱完了時に凍結している場合は、効果を発動せずCDのみ開始する。
